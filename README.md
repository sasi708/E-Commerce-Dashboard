E-Commerce Dashboard (Power BI)

This project presents an interactive E-Commerce Dashboard developed using Power BI to analyze key business metrics such as sales performance, revenue trends, delivery time, customer behavior, and website engagement.

The dashboard includes dynamic filters for date range, device type, city, product category, and gender, allowing users to explore data from multiple perspectives. Visualizations such as KPI cards, bar charts, line charts, and pie charts provide clear insights to support data-driven business decisions and performance monitoring.

Key Features:

Sales and revenue analysis by month

Delivery time and session duration overview

Customer behavior and returning customer analysis

Device type, city, product category, and gender filters

Interactive and user-friendly dashboard design

Tool Used: Power BI
